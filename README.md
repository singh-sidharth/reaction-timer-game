# Reaction Timer Game
A minigame that measure your click reaction time in 10 ms intervals.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```

```
